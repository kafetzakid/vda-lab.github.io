## Setup of the course
Parts:

* General introduction data visualization
* Visual design workshop
* Introduction to P5 (= javascript visualization library)
* Project work

## Evaluation

Evaluation will be based on 2 projects. Using data from experts or open datasets, you will create 2 visuals:

* a static visual to convey a message hidden in the data
* an interactive visualization that allows experts to explore their data
